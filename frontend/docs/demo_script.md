# PentestAI Frontend Demo Script

## 30s 快速讲解路径

1. 进入 `Workbench`，展示 `Task / Skills / Prompt` 三个治理入口。
2. 新建任务时选择 `自动` 或 `人工审核`，强调可控执行。
3. 进入任务详情，展示 `Decision Chat`：LLM 决策说明、PoC/Payload、审批动作。
4. 切到 `报告`，一键进入编辑态，实时预览 Markdown 渲染结果。
5. 返回 `Dashboard`，展示比赛驾驶舱指标与最近关键事件时间线。

## 60s 深入讲解路径

1. `TaskWorkbench`: 展示运行中任务、待审批任务和 exploit 成功率。
2. `SkillsWorkbench`: 展示技能启停开关、命中率、最近失败原因。
3. `PromptWorkbench`: 展示系统提示词版本、激活版本与回滚按钮。
4. `TaskDetail`: 手动审批一次，证明系统不是黑箱自动攻击。
5. `ReportCenter`: 对 AI 生成报告做人工修订，强调“人机协同闭环”。

## 推荐答辩话术

- “这不是单点工具，而是一个可治理的 AI 渗透工作台。”
- “我们把每一步 AI 决策可视化，既提高可信度，也降低误操作风险。”
- “报告支持在线编辑与实时渲染，保证结果可交付、可复核、可追溯。”
